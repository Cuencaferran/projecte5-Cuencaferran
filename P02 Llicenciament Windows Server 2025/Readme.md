# P02: Llicenciament Windows Server 2025 (Cas TransLògic S.A.)

## 📝 Breu Descripció

Mentre s'inicia l'aventura emprenedora de l'equip, la consultora *EverPia* ofereix el seu suport cedint encàrrecs de clients reals[cite: 1]. En aquest cas, l'empresa **TransLògic S.A.**, dedicada a la logística regional, necessita una proposta de consultoria per renovar i virtualitzar la seva infraestructura de servidors obsoleta cap a la nova versió de **Windows Server 2025** per tal de millorar-ne la disponibilitat[cite: 1].

---

## 🏢 Detalls de la Infrastructura del Client

Per poder realitzar l'estudi econòmic i tècnic, es disposa de les següents dades clau del client[cite: 1]:

### 🎛️ 1. Servidor Físic (Host)
* **Unitats:** 1 servidor físic[cite: 1].
* **Processadors:** 2 CPUs físiques[cite: 1].
* **Nuclis (Cores):** 12 nuclis físics per processador[cite: 1].
* **Total de nuclis a llicenciar:** 24 nuclis físics[cite: 1].

### 🖥️ 2. Càrrega de Treball (Màquines Virtuals - VMs)
L'empresa vol desplegar un **total de 12 Màquines Virtuals** amb Windows Server, distribuïdes de la següent manera[cite: 1]:
* `1 VM` per a Controlador de Domini (*Active Directory*)[cite: 1].
* `1 VM` per a Servidor de Fitxers[cite: 1].
* `1 VM` per a Servidor d'Impressores i Gestió Documental[cite: 1].
* `1 VM` per a SQL Server (Base de dades de l'ERP)[cite: 1].
* `8 VMs` de suport per a aplicacions de logística i terminals de magatzem[cite: 1].

### 👥 3. Usuaris i Dispositius
* **Plantilla total:** 45 empleats[cite: 1].
* **Personal d'oficina:** 30 treballadors (disposen de PC fix i ordinador portàtil cadascun)[cite: 1].
* **Personal de magatzem:** 15 mossos de magatzem (treballen en 3 torns rotatius i comparteixen un total de 5 tauletes robustes per fer l'inventari)[cite: 1].

---

## 📋 Encàrrec del Client (Tasques a Realitzar)

Com a consultors informàtics, s'ha de resoldre i documentar els següents punts principals[cite: 1]:

1. **Anàlisi del Model de Llicenciament:** Estudiar les regles de llicenciament per nucli (*core*) que aplica Microsoft a Windows Server 2025[cite: 1].
2. **Càlcul de Costos de l'SOP:** Calcular i comparar el cost total de les llicències base del servidor segons les dues opcions principals: **Windows Server Standard** i **Windows Server Datacenter**[cite: 1].
3. **Optimització de CALs:** Determinar quin tipus de llicència d'accés de client (CAL), si la **CAL d'Usuari (User CAL)** o la **CAL de Dispositiu (Device CAL)**, resulta més econòmica i eficient per a l'estructura de personal de TransLògic S.A.[cite: 1].
4. **Justificació Tècnico-Econòmica:** Argumentar de forma sòlida la decisió final basant-se en els costos obtinguts, la capacitat d'escalabilitat futura i el requeriment de funcionalitats avançades (com *Storage Spaces Direct*, *Software Defined Networking - SDN*, etc.)[cite: 1].
5. **Presentació Executiva:** Dissenyar una presentació (durada de 5-10 minuts) orientada a defensar la solució adoptada davant d'un perfil no tècnic (el gerent de TransLògic S.A.) de forma propera i entenedora[cite: 1].
