# P01: Control de Versions — GitHub Classroom i Git

## 📝 Breu Descripció

Fins ara, GitHub ha estat la plataforma central per presentar la documentació dels projectes[cite: 1]. Malgrat tot, documentar directament des de la interfície web presenta inconvenients de lentitud, especialment en gestionar carpetes o penjar imatges[cite: 1]. 

En aquest projecte fem un pas endavant en el món del control de versions: a partir d'ara **treballarem bàsicament en local**, aprofitant tota la potència del protocol `git`, i sincronitzarem el codi i la documentació local amb el repositori remot de GitHub[cite: 1].

---

## 🔒 Seguretat i Confidencialitat: Repositoris Privats

Per solucionar el problema dels repositoris públics —on no es pot controlar si algú s'aprofita del vostre treball—, implementem l'ús de **repositoris privats**[cite: 1]. D'aquesta manera s'assegura la total confidencialitat de la feina de l'equip[cite: 1].

### 🏫 El Model GitHub Classroom
Treballarem mitjançant **GitHub Classroom**, una plataforma educativa que simula l'estructura de les organitzacions empresarials reals[cite: 1]:
* Els repositoris són propietat de l'organització[cite: 1].
* La plataforma crea automàticament un *fork* privat i propi per a cada membre de l'equip[cite: 1].
* **El teu Portafoli Futur:** Més endavant, aquests repositoris passaran a ser públics de manera que podràs fer-ne un *fork* cap al teu compte personal de GitHub i afegir aquest projecte al teu portafoli professional[cite: 1].
