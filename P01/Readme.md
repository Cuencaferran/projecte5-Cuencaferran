🧑‍💻 P01: Control de versions
🐙 GitHub Classroom i git
📝 Breu descripció

Al llarg dels diferents projectes que hem anat treballant, GitHub ha esdevingut l’espai on presenteu la documentació 📂.
Tot i això, documentar directament des de la web té alguns inconvenients clars:

⚠️ Inconvenients de treballar només des del navegador

🐢 Lentitud en l’edició

📁 Crear carpetes és feixuc

🖼️ Penjar imatges i gestionar arxius no és gaire eficient

🚀 Nou enfocament: treballar com professionals

En aquest projecte farem un pas més en el món del control de versions:

💻 Treballarem principalment en local

🔁 Farem versionat amb git

☁️ Sincronitzarem el repositori local amb GitHub

Això ens permetrà aprofitar tota la potència de git i treballar de manera molt més àgil i realista.

🔒 Privacitat i seguretat dels repositoris

Fins ara, els vostres repositoris eren públics, cosa que fa impossible controlar si algú decideix aprofitar-se de la vostra feina 😕.

👉 La solució són els repositoris privats:

🔐 Només visibles per usuaris autoritzats

🛡️ Asseguren la confidencialitat del vostre treball

🏫 Com hi treballarem?

Utilitzarem GitHub Classroom, una plataforma pensada per a l’àmbit educatiu 🎓 que funciona de manera molt similar a les organitzacions professionals.

🔧 Funcionament de GitHub Classroom

🏢 Els repositoris són propietat d’una organització

🍴 Cada alumne té un fork automàtic del repositori base

🔒 Cada repositori és propi i privat

👨‍💻 Cadascú treballa de manera independent

🌍 Mirant cap al futur

Més endavant:

🔓 Els repositoris passaran a ser públics

👤 Podreu fer un fork al vostre GitHub personal

📁 Afegireu aquest projecte al vostre portafoli professional
